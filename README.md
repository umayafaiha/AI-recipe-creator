An AI-powered recipe generator using Claude (Anthropic).

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Copy `.env.example` to `.env` and add your API key
4. Run: `npm start`
5. Open http://localhost:3000

## Get API Key

Get your free Anthropic API key: https://console.anthropic.com/settings/keys

## Technologies

- Node.js + Express
- Anthropic Claude API
- Vanilla JavaScript
